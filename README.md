# Strategiehandbuch App. Peter & Martina Kücken

Diese Version ergänzt den bestehenden Peter-Bereich um einen eigenen Martina-Bereich.
Der Peter-Bereich bleibt bewusst unverändert.

## Dateien

- `index.html`
- `styles.css`
- `data.js`
- `app.js`

## GitHub Pages Nutzung

1. Neues GitHub Repository erstellen oder bestehendes Repository öffnen.
2. Diese vier Dateien in das Repository hochladen.
3. Bei GitHub unter `Settings` → `Pages` die Veröffentlichung aktivieren.
4. Als Quelle `Deploy from a branch` wählen.
5. Branch `main` und Ordner `/root` wählen.
6. Speichern.

## Funktionen

- Bereich Martina mit Band 1 bis Band 5
- 52-Wochen-Jahresprogramm
- LinkedIn-Beitrag pro Woche
- Facebook-Beitrag pro Woche
- WhatsApp-Status Montag bis Sonntag
- Veröffentlichungs-Häkchen pro Woche
- Notizfeld pro Woche
- Fortschrittsanzeige
- Suche in Martinas Bereich
- Hell-/Dunkelmodus
- Speicherung im Browser über LocalStorage

## Hinweis zur Integration

Wenn bereits ein Peter-Bereich existiert, ersetze diesen nicht. Kopiere nur Martinas Daten und die Navigationserweiterung in dein bestehendes Projekt.
