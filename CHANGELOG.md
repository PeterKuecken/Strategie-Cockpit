# Changelog

## Version 1.3.3
- Meine Kontakte anzeigen öffnet nur die alphabetische Kontaktliste.
- Es wird kein erster Kontakt mehr automatisch geöffnet.
- Kontaktliste zeigt weiterhin Name, Firma, Ort, ID, Status und nächsten Schritt.

## Version 1.3.2

- Button „Meine Kontakte anzeigen“ öffnet direkt die Adressdatenbank.
- Kontaktliste wird dort alphabetisch von A bis Z sortiert.
- Erster Kontakt der Liste wird automatisch geöffnet.
- Kontakt-ID wird auf fünf Stellen umgestellt, zum Beispiel PK-00001 oder MK-00001.
- Bestehende sechsstellige IDs werden beim Laden auf fünf Stellen normalisiert.
- In der Kontaktakte steht zuerst der Name, darunter die Kontakt-ID.

## Version 1.3.1

- Automatische PK/MK-ID beim Speichern.
